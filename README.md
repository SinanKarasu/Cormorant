# Cormorant (Modern Port)

Modernization of the original Cormorant project by Austin Zheng:
<https://github.com/austinzheng/Cormorant>

## Project Split

- Library: CormorantCore (Swift Package)
- Sample app: CormorantApp (SwiftUI)
- EPL-1.0 license preserved
- Swift 6.3 port

## Current Validation

This is currently smoke-tested with:

```clojure
(+ 2 3)
```

Expected result:

```text
5
```

Additional behavior is not yet fully re-verified in this fork.
